# MA Marketplace — Public Description

MA Marketplace is an early-access, local-first marketplace layer built on top of a trust-oriented infrastructure.

The goal is not to imitate large platforms.
The goal is to explore a model where:
- identity can be wallet-based
- events can be verifiable
- publishing can be signed
- activity can remain modular and user-controlled

This version should be treated as a pilot / early infrastructure release.

## Current expectations
- early access
- limited product set
- manual support where needed
- iterative improvements during launch

## First product strategy
Start with a small curated set:
- 3 to 5 strongest products
- clear photos
- simple status labels: available / reserved / sold
- short descriptions
- one clear path to contact or order
