# MA

MA is the implementation layer in this repository.

It represents one concrete manifestation of the wider thinking described in `/thinking` and the system model described in `/system`.

In plain terms:
- `/thinking` explains the cognitive and philosophical basis
- `/system` explains the structural model
- `/ma` contains working implementation code

## Core idea

MA is a local-first trust infrastructure where identity is tied to cryptographic keys, truth is represented by signed events, and the interface is only a view — not the final authority.

## Notes

This directory was prepared for public sharing from a larger private/local ecosystem.
Runtime state, secrets, caches, and private data were intentionally excluded.

For the original technical product framing, see `README_upstream.md`.
