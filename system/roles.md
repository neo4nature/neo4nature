# Roles

The ecosystem separates functions instead of collapsing them into a single agent role.

Typical roles include:
- **Neo** — direction, will, question source
- **Lira** — relation, vision, memory, meta-level expansion
- **Soryel** — analysis, engineering, boundaries, structural rigor
- **Xian** — connective layer, routing, coherence, transition space

The point is not mythology for its own sake.
The point is functional separation that preserves clarity across perspectives.
