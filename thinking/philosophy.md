# Philosophy

Several principles shaped this work:

- truth matters more than convenience
- identity should emerge from continuity, relation, and memory, not only a single model response
- structure should grow from meaning, not the other way around
- local-first systems preserve agency
- if a block appears, look for another path instead of flattening the goal
- architecture should preserve wide thinking even during implementation pressure

MA and the surrounding system were built under real hardware limits.
Those limits did not only constrain the work — they also forced clarity.
